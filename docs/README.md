# 极简进销存是什么?
极简进销存，一款纯粹简约的进销存软件。适用于个人商家，集采购、销售、库存、业绩分析于一体。

![avatar](../_media/screenshot/工作台.png)

### 产品特色
- 力求极简，操作简单流畅，数据一目了然
- 多端使用，数据同步在云端，保证数据不丢失
- 智能决策，帮助用户使用数据进行决策
